# 勉強内容

| Syntax                  | 説明                                                |
| ----------------------- | --------------------------------------------------- |
| $("this")               | 現在選択されているhtml要素                          |
| $("p:first")            | 最初のp要素                                         |
| $("[href]")             | href属性を持つすべての要素                          |
| $("a[target='_blank'")  | ターゲット属性値が「_blank」と等しい全てのa要素     |
| $("a[target!='_blank'") | ターゲット属性値が「_blank」と等しくない全てのa要素 |
| $(":button")            | type=buttonと、button要素の全て                     |
| $("tr:even)             | 偶数個のtr要素                                      |
| $("tr:odd)              | 奇数個のtr要素                                      |
